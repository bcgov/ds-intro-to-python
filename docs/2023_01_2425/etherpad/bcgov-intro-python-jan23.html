January 25 - Cleaning Data

https://raw.githubusercontent.com/bcgov/ds-intro-to-python/main/data/movieratings.csv

To get rows with missing data:

where_na = movie_ratings.isna().any(axis=1)
missing_data = movie_ratings[where_na]
missing_data.head()

the .isna() function tells us True/False for every value in the dataframe
the .any() function tells us if there is at least one True in the dataframe
the optional argument axis=1 tells us to look across each row 

axis{0 or ‘index’, 1 or ‘columns’}
Axis along which to fill missing values. For Series this parameter is unused and defaults to 0.

To only remove a row if both finding nemo AND parasite is missing:
    
where_nemo_parasite_missing = (
    movie_ratings
    .isna()[['Finding Nemo', 'Parasite']]
    .all(axis=1)
)
nemo_parasite = movie_ratings[~where_nemo_parasite_missing]

this is similar to the first one, but first we reduce our dataset to only looking at the columns we want 
(Stu will be covering this!)
We then use the .all() instead of .any() function
the .all() function tells us True if everything is True, False otherwise
axis = 1 tells us to look across both columns
Next we take the NEGATION (using the ~) of the missing data, because we actually want the non-missing rows 

in both cases, we are using the square brackets [] as a way of 'slicing' a dataframe we haven't come across yet
the objects where_na and where_nemo_parasite_missing are both series consisting of bools
it's a list something like [True, True, False, False, True, ... etc]
so when we do movie_ratings[ [True, False, True, True, ...] ] it keeps only the rows where there was True

doing them all at once: https://bcgov.github.io/ds-intro-to-python/32_advanced_pandas.html#multiple-conditions

January 25 - Using Pandas

regarding index vs 'ID' columns: the pandas index is a special column that belongs to each series, and is a separate notion from ID. If we wanted to, we can merge two tables together specifically using the index, but it is not necessary. In the merge function, we would use the parameters:
    
    df_left.merge(df_right, left_index=True, right_index=True)
    
to use the index columns as our joining key.
